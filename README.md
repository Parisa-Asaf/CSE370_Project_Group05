# CSE370_Project_Group05
BracU Hub is  a web-based community platform for BRACU students with material sharing, user profiles, and Q&amp;A features. SQL project.
Project Proposal: BracU Hub / BracU Learn

1. Project Objectives

To create a secure and user-friendly portal for BRACU students.
To facilitate knowledge sharing through personal material uploads.
To promote peer-to-peer learning using a community-based Q&A system.
To enhance academic communication and resource availability.
To ensure data privacy and permission-based sharing of materials.


2. Key Features (6 Features)

Feature 1: User Registration and Login System
Secure authentication using PHP and MySQL.
Password encryption and session management.
Feature 2: User Profile Management
Each student can create and edit their profile.
Includes profile photo, department, semester, and brief bio.
Option to update status posts (like mini social updates). Optional
Feature 3: Material Upload and Management
Students can upload study materials (PDFs, notes, presentations, etc.).
Option to make materials:
Private (accessible only by the uploader)
Public (accessible by all registered users)
File upload validation and organized material listing.
Feature 4: Access Control & Permissions
Material access is controlled based on user permissions.
Owner can toggle privacy (Private / Public l).
Ensures secure handling of student-generated content.
Feature 5: Community-Based Q&A System
Students can post questions and answer othersâ€™ queries.
Categories for subjects/courses.
Option for upvotes/downvotes to highlight useful answers.
Feature 6: Search bar to find users, materials, or Q&A posts.
Optional 7: Leaderboard for top contributor.
